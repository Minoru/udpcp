# Требования

- Linux x86_64
- gcc >= 4.9

# Сборка

```
$ make
```

# Запуск

```
$ make run -j1
```

Сервер завершит работу спустя 5 секунд после того, как закончится передача
последнего файла.

Можно указать больше потоков; в этом случае несколько клиентов будут
одновременно отправлять серверу разные файлы.
